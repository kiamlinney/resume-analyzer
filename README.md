An enterprise-ready serverless AI resume analyzer that leverages LLMs (GPT/Claude) to provide real-time ATS feedback and scoring logic.
- Designed and implemented a React + TypeScript front end with Tailwind CSS v4 
- Integrated Puter.js for a fully serverless architecture supporting authentication, file storage, and AI directly in the browser
- Employed Zustand for global state management, React Router v7 for navigation, and Docker for production deployment
- Utilized LLM APIs to generate intelligent resume insights and scoring logic
- Gained advanced experience in scalable front-end architecture, cloudless API integration, and AI-driven UX design through iterative development and testing
